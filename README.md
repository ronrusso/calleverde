# Calle Verde

Calle Verde is an AI-first media and cultural education platform focused on preserving, modernizing, and monetizing Latino heritage through scalable digital IP.

This repository hosts the initial static website deployment for CalleVerde.com, including the published business plan and positioning materials.

## Purpose
Calle Verde exists to build culturally authentic, ownable media properties for the global Latino audience using AI-driven storytelling, avatar media, and automated content pipelines.

## Contents
- `index.html` — Root entry point for the public site (GitHub Pages)
- Static HTML deployment generated from the Calle Verde business plan

## Deployment
This site is deployed using GitHub Pages and is intended to be mapped to the custom domain:
- https://calleverde.com

Initial deployment.
